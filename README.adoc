== Convertir le projet d'AsciiDoc vers HTML

- Installer link:https://rubygems.org/pages/download[gem].
- Installer link:http://bundler.io/[bundler] :

 gem install bundler

- Installer les packages :

 bundle install

- Convertir le projet en HTML

 make index.html -B
